# Expense Tracker

Simple browser-based expense tracker.

## What it does

- Lets you add expenses with title, amount, category, date, and notes
- Saves data in browser storage so entries stay there after refresh
- Exports a `.csv` file that opens in Excel or imports into Google Sheets

## How to use it

1. Open `index.html` in your browser.
2. Fill in the form and save expenses.
3. Click `Download CSV` when you want a spreadsheet file.
